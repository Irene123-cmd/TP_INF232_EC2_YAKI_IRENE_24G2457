# 🏥 SantéCollect

Plateforme de collecte, analyse et visualisation de données médicales.

## 📋 Fonctionnalités

- Formulaire structuré en 6 sections
- Validation en temps réel (IMC, champs obligatoires)
- Tableau de bord avec graphiques interactifs
- Export CSV / JSON / PDF
- Mode hors-ligne (stockage localStorage)
- Design responsive

## 🚀 Utilisation

1. Ouvrir `index.html` dans un navigateur
2. Remplir des fiches patients via le formulaire
3. Visualiser les données dans le tableau de bord
4. Exporter les données pour analyse externe

## 📦 Technologies

- HTML5 / CSS3
- Chart.js (visualisation)
- localStorage (persistance)
- Vanilla JavaScript

## 🔒 Sécurité

Les données sont stockées localement dans le navigateur.
Pour la production, utiliser une base de données sécurisée.
